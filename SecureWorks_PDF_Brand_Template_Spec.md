# SecureWorks WA — PDF Brand Template Specification

## Purpose
This document defines the visual rules for EVERY PDF generated by the patio configurator. All PDF generators (quote, work order, steel order, sheets order, fab order, and any future documents) must follow these rules to maintain brand consistency.

This is a SPECIFICATION ONLY — not code. The developer should reference this when building or modifying any PDF generation function.

---

## Brand Colours (exact values)

| Name | HEX | RGB | Use |
|------|-----|-----|-----|
| SecureWorks Orange | #F15A29 | 241, 90, 41 | Header band, accent lines, highlights |
| Dark Dusty Blue | #293C46 | 41, 60, 70 | Section header bars, heading text, dark panels |
| Mid Dusty Blue | #4C6A7C | 76, 106, 124 | Column header text, field labels, secondary text |
| White | #FFFFFF | 255, 255, 255 | Backgrounds, text on dark elements |
| Table Row Tint | — | 240, 244, 247 | Alternating table row background (very light blue-grey derived from Mid Dusty Blue) |

### Colours NOT to use
- No pure black (#000000) for headings — use Dark Dusty Blue instead
- No generic greys for section panels or backgrounds
- No orange as a section/panel background fill (orange is for the header band and thin accents only)
- Body text can be near-black: RGB 50, 50, 50 or similar

## Page Layout

### Margins
- Top: 0mm (header band goes full width to edge)
- Left: 20mm
- Right: 20mm
- Bottom: 25mm (footer space)

### Header Band (every page)
- Full width, height: 28-32mm
- Background: SecureWorks Orange (#F15A29)
- Left side: "SecureWorks WA" in white, bold, 20pt
- Below company name: "Patios • Carports • Pergolas" in white, normal, 9pt
- Right side: Job number in white, bold, 14pt
- Below job number: Date in white, normal, 9pt
- No logo image required (text-only header is acceptable, but if logo is embedded as base64, use the white version)

### Footer (every page)
- Thin line: Orange (#F15A29), 0.5pt weight, full width at ~275mm from top
- Below line, centred: "SecureWorks WA | Sales: [salesperson name]" in Mid Dusty Blue, 8pt
- Below that, centred: "Page X of Y | [job number]" in Mid Dusty Blue, 7pt

## Component Styles

### 1. Page Title (one per page, below header)
- Text: Dark Dusty Blue (#293C46), bold, 16-18pt
- Orange accent underline: 0.8pt line, Orange (#F15A29), from left margin to ~60mm wide
- Spacing: 8mm below header band, 6mm below underline before content starts

### 2. Section Header Bars
- Full content width (left margin to right margin)
- Height: 8-9mm
- Background: Dark Dusty Blue (#293C46)
- Left edge accent: 3px Orange (#F15A29) stripe on left side of bar
- Text: White, bold, 10-11pt, vertically centred, 4mm left padding after orange stripe
- Spacing: 4mm above, 3mm below before section content

### 3. Tables
- Column headers: Mid Dusty Blue (#4C6A7C) text, bold, 9pt, on white background with a 0.5pt bottom border in Mid Dusty Blue
- Data rows: alternating White and Table Row Tint (240, 244, 247)
- Text: near-black (50, 50, 50), normal, 9pt
- Row height: 7-8mm
- Cell padding: 3mm left
- No heavy borders — use 0.3pt lines in light grey (220, 220, 220) if needed for separation
- Right-align numerical values (quantities, lengths, prices)

### 4. Detail/Spec Panels (for construction details, key dimensions, etc.)
- Background: Table Row Tint (240, 244, 247)
- Border: none (or 0.3pt in 220, 220, 220 if needed)
- Corner radius: 2mm
- Padding: 6mm all sides
- Label text: Mid Dusty Blue (#4C6A7C), normal, 8-9pt
- Value text: Dark Dusty Blue (#293C46), bold, 9-10pt

### 5. Image Cards (for 3D renders, elevation drawings, truss diagrams)
- Background: Table Row Tint (240, 244, 247)
- Corner radius: 2mm
- Padding: 4mm around image
- No border, or 0.3pt light grey

### 6. Checklists
- Checkbox: 4×4mm square, 0.5pt border in Mid Dusty Blue
- Check mark (if pre-checked): Orange (#F15A29)
- Label text: near-black, normal, 9pt
- Row spacing: 6-7mm

### 7. Signature/Sign-off Blocks
- Section header bar (same as #2 above)
- Signature lines: 0.5pt, Mid Dusty Blue
- Label text ("Installer:", "Date:"): Mid Dusty Blue, normal, 9pt
- Confirmation text: italic, Mid Dusty Blue, 8pt

### 8. Highlight/Call-out Boxes (for important notes, warnings)
- Background: Orange (#F15A29) at 8% opacity (or RGB 253, 240, 236)
- Left border: 3px solid Orange (#F15A29)
- Text: Dark Dusty Blue, normal, 9pt
- Padding: 5mm all sides

### 9. Investment/Price Block (quote PDF only)
- Background: Dark Dusty Blue (#293C46)
- Corner radius: 2mm
- Label: White, normal, 10pt
- Price: Orange (#F15A29), bold, 22pt
- Sub-text (GST note): White at 70% opacity, normal, 8pt

## Document-Specific Notes

### Quote PDF
- Page 1: Hero render + two secondary renders in image cards
- Page 2: Specifications panel, inclusions with orange ticks, investment block, payment schedule, acceptance/signature, terms
- Customer-facing — most polished

### Work Order
- Multiple pages — use consistent header/footer throughout
- Construction detail pages: render on left (image card), specs on right (detail panel)
- Installation sequence: numbered steps with section header style for step groups
- Materials checklist: table with checkbox column
- Keep sign-off as final page

### Steel Order / Sheets Order / Fab Order
- Single page each (typically)
- Main order table using standard table style
- Nesting summary (steel order): use detail panel style
- Flashing sketches area (sheets order): empty bordered boxes using 0.3pt Mid Dusty Blue border
- Truss diagram (fab order): use image card style, make render at least 50% page width for readability
- Truss specs (fab order): use detail panel style with label/value pairs — NOT orange background sections

---

## What NOT To Do
1. Do NOT use orange as a background fill for content sections (only for the header band)
2. Do NOT use generic grey — use the brand blue tones instead
3. Do NOT vary the header band between document types
4. Do NOT use different table styles between documents
5. Do NOT put cutting instructions in item name fields — use Notes column
6. Do NOT mix font sizes arbitrarily — stick to the defined size ranges
